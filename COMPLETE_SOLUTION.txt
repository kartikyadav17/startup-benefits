## ğŸ†˜ TROUBLESHOOTING

### Problem: Port 3000 or 5000 already in use
**Solution:**
```powershell
Get-Process node | Stop-Process -Force
```

### Problem: npm packages missing
**Solution:**
```powershell
cd startup-benefits-backend && npm install
cd ../startup-benefits-frontend && npm install
```

### Problem: MongoDB not running
**Solution:** Edit `startup-benefits-backend\.env`:
```env
MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/startup-benefits
```
(Use MongoDB Atlas for cloud hosting)

### Problem: Build fails
**Solution:**
```powershell
cd startup-benefits-backend && npm run build
cd ../startup-benefits-frontend && npm run build
```
